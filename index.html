SpoinkOS

Welcome to SpoinkOS

SpoinkOS is a lightweight, bloatware-free Linux distribution based on Ubuntu. We've stripped down Ubuntu to give you a clean and developer-friendly environment.

Features

  • User-Friendly Interface: Our distro comes with a clean and intuitive desktop environment called GNOME that makes it easy to use, even for newcomers.
  • Based On Ubuntu: SpoinkOS is built on the solid foundation of Ubuntu, ensuring compatibility and stability.
  • Developer-Friendly: SpoinkOS provides tools and libraries tailored for developers, making it an ideal platform for coding and testing.

Installation

  1. Download the ISO image from the Internet Archive mirror.
  2. Create a bootable USB drive using your preferred tool (e.g., Rufus, Etcher).
  3. Boot your computer from the USB drive.
  4. Follow the on-screen instructions to install SpoinkOS.
  5. (For Developers) To make a NixOS-Like deployment system, first install the OS on hardware or a VM then use CLCTool to execute the deployment / config scripts you have written; More info at CLCTool

Conclusion

Thank you for choosing SpoinkOS! We hope you enjoy using our distro as much as we enjoyed creating it!

Happy computing!
