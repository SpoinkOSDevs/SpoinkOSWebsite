SpoinkOS

Welcome to SpoinkOS

SpoinkOS is a lightweight, bloatware-free Linux distribution based on Ubuntu. Enjoy a clean and developer-friendly environment.

Features

  • User-Friendly Interface: SpoinkOS features a clean and intuitive GNOME desktop environment.
  • Based On Ubuntu: Built on the solid foundation of Ubuntu for compatibility and stability.
  • Developer-Friendly: Tailored tools and libraries make SpoinkOS ideal for coding and testing.

Installation

  1. Download the ISO image from the Internet Archive mirror.
  2. Create a bootable USB drive using your preferred tool (e.g., Rufus, Etcher).
  3. Boot your computer from the USB drive.
  4. Follow on-screen instructions to install SpoinkOS.
  5. (For Developers) To make a NixOS-Like deployment system, install the OS, then use CLCTool for deployment / config scripts; More info at CLCTool

Conclusion

Thanks for choosing SpoinkOS! We hope you enjoy using it as much as we enjoyed creating it!
